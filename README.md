bootstrap3-steps
================

Create bootstrap style steps

Use
===

```html
<ul class='steps'>
	<li class='step'>
		<a href="first">First step</a>
	</li>
	<li class='step'>
		<a href="second">Second step</a>
	</li>
	<li class='step'>
		<a href="third">Third step</a>
	</li>
</ul>
```
